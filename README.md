# VE社区 公开仓库
1. 介绍
    - 此仓库仅用于存放配置文件, 以及公开头文件.
    - 任何想进行推送以用于合并分支的, 将会被检查以确保内容可以被部署到所有服务器.
2. 文件结构
    - stripper
        - 将会被部署到全部服务器, 如果该服务器上有对应地图, 将会自动启用对应的配置.
